numpy.ma.masked_array.iscontiguous#

method

ma.masked_array.iscontiguous()[source]#

Return a boolean indicating whether the data is contiguous.

Parameters:
None

Examples

>>> x = np.ma.array([1, 2, 3])
>>> x.iscontiguous()
True

iscontiguous returns one of the flags of the masked array:

>>> x.flags
  C_CONTIGUOUS : True
  F_CONTIGUOUS : True
  OWNDATA : False
  WRITEABLE : True
  ALIGNED : True
  WRITEBACKIFCOPY : False
